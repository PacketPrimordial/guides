Lunar V2 – Installation Guide

This guide is intentionally simple and direct. Follow the steps in order.

1. Install Python 3.10.5

Lunar V2 requires Python 3.10.5.

Download it here:

https://www.python.org/downloads/release/python-3105/

Important: During installation, check “Add Python to PATH”.

2. Purchase Lunar V2

Lunar V2 can be purchased from the official website:

https://gannonr.com

After purchase, download the Lunar V2 files provided to you.

3. Install PyTorch

PyTorch is required for Lunar V2.

Go to the official PyTorch site and select your system configuration:

https://pytorch.org/get-started/locally/

Example (CPU only):

pip install torch torchvision torchaudio

4. Optional Hardware (Advanced / USB A Devices)

Some users choose to use a compatible USB A microcontroller for advanced use cases.

Recommended options:

  • Adafruit RP2040 (USB A port)
  • Adafruit RP2350 (USB A port)

This step is optional and only needed for specific setups.

5. Final Notes

  • Make sure Python 3.10.5 is the active version.
  • Install all required dependencies before running Lunar V2.
  • Follow any additional instructions included with your Lunar V2 download.
